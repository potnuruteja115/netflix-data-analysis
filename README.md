# Netflix Data Analysis

## Project Overview
Performed Exploratory Data Analysis (EDA) on Netflix dataset to analyze content trends, genre distribution, and yearly growth.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- Identified top content-producing years.
- Analyzed most popular genres.
- Visualized trends using data visualization techniques.

## Conclusion
Derived meaningful insights to understand content growth patterns.
